This program is designed to get each individual character from a given input file print that token and also
print out what kind of token it is.

How to Run:


In command line the user will type :
        "gcc tokenizer.c"
        "./a.out unix_input_txt output.txt"

Where a.out is the name of the executable , unix_input_txt is the input file and output.txt is the file
the user wishes to write to.

Known issues:
        At the moment the program doesnt handle combined tokens such as <= or numbers longer than one character
